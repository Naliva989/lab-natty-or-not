# Dominó de Figuras

## 📒 Descrição
O Objetivo é gerar um jogo de dominó que use figuras geometricas no lugar dos numeros

## 🤖 Tecnologias Utilizadas
Nesse projeto usei o o chat-GPT e Leonardo.Ai

## 🧐 Processo de Criação
Usei o Chat-GPT para gerar um propt que liste as figuras para gerar as imagens para o dominó e as regras para jogar e o Leonardo.Ai para gerar as imagens com os propts do chat-GPT

## 🚀 Resultados
O resultado foi um printable divertido para usar com as crianças de forma rapida no fim de semana.
